# SQL Subquery Practice Repository

This repository contains 25+ SQL Subquery questions with solutions.

The goal of this project is to strengthen SQL problem-solving skills through real-world business scenarios commonly asked in Data Analyst, Business Analyst, SQL Developer, and Data Engineering interviews.

## Topics Covered

- Single Row Subqueries
- Multiple Row Subqueries
- Correlated Subqueries
- Nested Subqueries
- EXISTS & NOT EXISTS
- IN & NOT IN
- ANY & ALL
- Scalar Subqueries
- Department-wise Analysis
- Salary Analysis

- Master SQL Subqueries
- Improve Query Optimization Thinking
- Solve Real Interview Questions
- Build Strong Analytical SQL Skills
